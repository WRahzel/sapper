== Инструкция

Для правильного запуска игры нужно что бы были установлены такие же версии:

* Ruby 2.2.3p173 

* Rails 4.2.6

1. Нужно открыть папку sapper-master через терминал <tt>сd sapper-master</tt>.
2. Устанавливаем гемы <tt>bundle install</tt>.
3. Создаем базу данных и запускаем миграции из папки db/migrate <tt>rake db:migrate</tt>.
4. Запускаем проект (игру) на веб-сервере <tt>rails s</tt>.
